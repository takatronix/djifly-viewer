Live Streaming Settings

Configure live streaming settings in the DJI Fly app.

  1. Open the settings menu in the DJI Fly app
  2. Select "Live Streaming"
  3. Select "Custom RTMP"
  4. Enter the following RTMP URL
rtmp://[This app's IP address]:1935/live/s

üí° How to Check IP Address
